## Clipped
This program generates a time array of numbers that represent a sin wave at a specific frequency in hertz. This frequency is determined by the user on running the program. The program then calculates, writes, and outputs a .wav file of a regular wave at the given frequency, as well a clipped version of it. Both waves are 1 second long.
